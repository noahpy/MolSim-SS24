\doxysubsection{Detailed Description}
Represents a 3D grid of cells to organize particles in a simulation. 

This class provides methods for adding particles, identifying neighboring particles, and iterating over boundary and halo cells. It also internally manages the domain size, cutoff radius, and grid dimensions.
