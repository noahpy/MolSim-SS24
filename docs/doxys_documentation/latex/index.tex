\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}{\ttfamily \doxylink{classSimulation}{Simulation}}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}
\begin{DoxyItemize}
\item Any kinds of a simulations are represented as a child class of the {\ttfamily \doxylink{classSimulation}{Simulation}} class
\item Any simulation instance defines the {\ttfamily run\+Sim()} function, which uses the I/O classes ({\ttfamily \doxylink{classFileReader}{File\+Reader}}, {\ttfamily \doxylink{classFileWriter}{File\+Writer}}), the {\ttfamily \doxylink{classPhysicsStrategy}{Physics\+Strategy}} class and the model variables passed to it to calculate the simulation through time.
\item Child simulations might extend with new model varibales their parent simulations.
\end{DoxyItemize}
