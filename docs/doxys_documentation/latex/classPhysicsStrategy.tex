\doxysubsection{Detailed Description}
Strategy pattern for physics calculations. 

This is a strategy pattern for physics calculations. This allows to specify which physics calculations to perform at runtime, by passing the function references. By passing the simulation object to the functions, the functions can use members which are only defined in subclasses of \doxylink{classSimulation}{Simulation}.
