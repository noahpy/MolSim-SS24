\doxysubsection{Detailed Description}
\doxylink{classSimulation}{Simulation} class for the Lennard-\/\+Jones simulation with a molecules. 

This class is a subclass of the \doxylink{classMixedLJSimulation}{Mixed\+LJSimulation} and is used to simulate different particles in one simulation, with an additional molecules.
