\doxysubsection{Detailed Description}
\doxylink{classSimulation}{Simulation} class for the Lennard-\/\+Jones simulation. 

This class is a subclass of the \doxylink{classSimulation}{Simulation} class and is used to simulate the movement of particles according to the Lennard-\/\+Jones potential, by utilizing the linked-\/cell approach.
