\doxysubsection{Detailed Description}
Represents a single cell within the \doxylink{classCellGrid}{Cell\+Grid}. 

A \doxylink{classCell}{Cell} stores particles and tracks its type (boundary, halo, or bulk). It provides methods for managing particles, accessing their references, and managing a neighbor counter.
