A factory function for creating FileWriter objects.

This function creates and returns a unique pointer to a FileWriter or one of its derived classes based on the file type provided.

Parameters
typeAn unsigned integer representing the type of file writer to create.
out_nameThe name of the output file.
Returns
A unique pointer to a FileWriter object or nullptr if the file type is not supported.
